## Python automation tools

To make setting up challenge projects easier, since I reuse many styles, README content, and tags in index.html, I created python scripts that will generate these as starter files. These scripts will generate both starter files for the challenge and update this README with the challenge to Solutions table. You can read more about the scripts [here](https://github.com/jordanallybrown/frontendmentor/tree/main/python-tools)
